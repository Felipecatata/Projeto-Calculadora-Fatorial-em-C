# Calculadora Fatorial em C

Projeto desenvolvido em linguagem C para simular uma calculadora que pega um número inteiro positivo inteiro e calcula seu fatorial. O objetivo foi praticar lógica de programação e estruturas de controle.

## Funcionalidades

- Cálculo do fatorial de um número inteiro
- Entrada de dados pelo usuário
- Exibição do resultado no terminal

## Tecnologias utilizadas

- Linguagem C
- VS Code
- Git/GitHub

## Conceitos praticados

- Lógica de programação
- Estruturas de repetição (loops)
- Funções (se aplicável)
- Entrada e saída de dados (scanf e printf)
- Operações matemáticas

## Como executar

gcc fatorial.c -o fatorial

./fatorial
